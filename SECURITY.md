# Security Policy

## Supported Versions

Security fixes are applied to the latest supported release line.

## Reporting a Vulnerability

Please do not open public issues for security-sensitive reports.

If GitHub private vulnerability reporting is enabled for this repository, use that channel.

Otherwise:

1. contact the maintainer through GitHub
2. provide a clear reproduction path
3. include impact, affected platform, and target CLI

Please allow reasonable time for triage before public disclosure.
